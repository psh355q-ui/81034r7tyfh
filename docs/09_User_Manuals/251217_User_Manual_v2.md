## 시스템 개요

AI Trading System은 글로벌 매크로 분석, AI 집단 토론, 추론 추적, 승인 워크플로우를 통합한 **지능형 투자 의사결정 플랫폼**입니다.

### 핵심 철학
- **안전 우선:** AI가 모든 결정을 하지만, 최종 승인은 사람이 합니다
- **투명성:** AI의 모든 추론 과정을 추적하고 기록합니다
- **집단 지성:** 5개의 독립적인 AI 에이전트가 토론하여 편향을 최소화합니다

## 주요 기능별 사용 시나리오

### 🌍 Global Macro (글로벌 매크로)

**언제 사용하나요?**

✅ **거시경제 환경 점검**
- 새로운 투자 시작 전 글로벌 리스크 체크
- 예: "지금 주식 투자하기 좋은 시기인가?"

✅ **국가별 리스크 비교**
- 해외 투자 국가 선택 시
- 예: "미국 vs 중국, 어디에 투자할까?"

✅ **글로벌 이벤트 영향 분석**
- Fed 금리 인상, 중국 PMI 하락 등의 나비효과 분석
- 예: "Fed가 금리 올리면 내 포트폴리오는?"

✅ **리스크 환경 확인**
- 보유 종목이 특정 국가/지역에 노출되어 있을 때
- 예: "지금 중국 투자 안전한가?"

**활용 팁:**
- 매주 월요일, 미국 장 개장 전 체크
- 중요한 글로벌 이벤트 발생 시 즉시 확인
- 국가별 리스크 점수 60점 이상이면 주의

---

### 🔍 Logic Trace (추론추적)

**언제 사용하나요?**

✅ **AI 결정의 근거 확인**
- AI가 추천한 매매 결정의 이유를 상세히 확인
- 예: "왜 NVDA를 사라고 했지?"

✅ **잘못된 판단 원인 분석**
- 과거 손실 거래의 원인 분석
- 예: "왜 그때 손절했어야 했는데 안 했지?"

✅ **감사/컴플라이언스 보고**
- 규제 기관 제출용 AI 결정 프로세스 문서화
- 예: "이 거래가 왜 합법적인지 증명"

✅ **AI 학습 패턴 이해**
- AI가 어떤 데이터를 중요하게 보는지 파악
- 예: "이 AI는 기술적 분석을 더 신뢰하나?"

**활용 팁:**
- 큰 금액 거래 전 반드시 확인
- 예상과 다른 AI 추천이 나왔을 때
- 매월 말, 투자 성과 복기 시 활용

### 🎭 War Room (AI 투자위원회)

**언제 사용하나요?**

✅ **중요한 큰 거래 결정**
- 포트폴리오의 10% 이상을 움직일 때
- 예: "전 재산의 30%를 TSLA에 몰빵?"

✅ **의견이 엇갈릴 때 토론**
- 기술적 분석과 펀더멘털 분석이 상반될 때
- 예: "차트는 매수인데 실적은 안 좋은데?"

✅ **다양한 관점 필요**
- 복잡한 시장 상황에서 다각도 분석
- 예: "반도체 섹터 전체를 어떻게 봐야 하나?"

✅ **고위험 종목 판단**
- 변동성이 큰 종목이나 테마주
- 예: "과연 지금 사야 하나?"

**War Room 에이전트 역할:**
- 🧑‍💻 **Trader:** 단기 매매 신호
- 👮 **Risk:** 리스크 관리 (보수적)
- 🕵️ **Analyst:** 펀더멘털 분석
- 🌍 **Macro:** 거시경제 환경
- 🏛️ **Institutional:** 기관 투자자 관점

**활용 팁:**
- 5개 에이전트 중 4개 이상 동의 시 신뢰도 높음
- Risk 에이전트가 반대하면 재검토 필요
- Constitutional AI 검증 실패 시 절대 진행 금지

---

### ✅ Approvals (승인 워크플로우)

**언제 사용하나요?**

✅ **AI 추천 거래 최종 승인**
- War Room 토론 결과를 사람이 검토
- 예: "AI가 매수하라는데 정말 괜찮을까?"

✅ **24시간 자동 승인 대기**
- Soft Approval (낮은 우선순위)
- 예: "24시간 내 반대 안 하면 자동 실행"

✅ **리스크 높은 거래 차단**
- Hard Approval (높은 우선순위)
- 예: "이건 무조건 내가 확인해야 해!"

**승인 레벨:**
- **NO_APPROVAL:** 자동 실행 (Priority < 30%)
- **SOFT_APPROVAL:** 24시간 후 자동 승인 (Priority 30-70%)
- **HARD_APPROVAL:** 사람 필수 승인 (Priority > 70%)

## 워크플로우 가이드

### 📌 일반적인 투자 의사결정 프로세스

```
1. Global Macro 체크
   ↓
   리스크 높음? → 거래 중단
   ↓
2. War Room 토론
   ↓
   합의 75% 미만? → 재검토
   ↓
3. Approval Queue
   ↓
   사용자 승인/거부
   ↓
4. 실거래 실행
   ↓
5. Logic Trace 저장
```

### 🔄 실전 예시: NVDA 매수 결정

**Step 1: Global Macro**
- 미국 리스크: 45.5점 (정상)
- Tech 섹터 상관관계: 양호
- ✅ 진행 가능

**Step 2: War Room**
- Trader: BUY 85%
- Risk: HOLD 65%
- Analyst: BUY 70%
- Macro: BUY 75%
- Institutional: BUY 80%
- **결과:** 4/5 BUY (합의 80%)

**Step 3: Approval**
- Priority Score: 88% (HARD_APPROVAL)
- AI 근거: "GTC 키노트 후 기관 매수 급증"
- ✅ 사용자 승인

**Step 4: 실행**
- NVDA 50주 매수 @ $875.50

**Step 5: Logic Trace**
- 추론 트리 저장
- 향후 복기 가능

## FAQ

**Q: War Room과 Logic Trace의 차이는?**  
A: War Room은 **5개 AI의 토론** (다양성), Logic Trace는 **1개 AI의 추론** (투명성)

**Q: Global Macro는 실제 데이터인가요?**  
A: 조건부 실제 데이터. CountryRiskEngine 사용 가능 時 실제, 아니면 Mock.

**Q: Approval을 건너뛸 수 있나요?**  
A: Priority 30% 미만은 자동 실행 (NO_APPROVAL). 나머지는 승인 필요.

**Q: 24시간 자동 승인이 불안해요.**  
A: Soft Approval 설정을 Hard로 변경하거나 즉시 거부하세요.

**Q: War Room에서 5개가 의견이 다르면?**  
A: 합의 75% 미만이면 재검토 권장. Risk 에이전트 의견 특히 중요.
