### 왜 모두 거부되었나?
헌법이 **정상적으로 작동**하여 위험한 거래를 모두 차단했습니다:

**거부 사유**:
1. **제1조 위반 (자본 보존)**
   - 단일 주문 크기가 자본 대비 너무 큼 (>10% 제한)
   - 예: AAPL 100주 x $150 = $15,000 (자본의 15% > 10% 한도)

2. **제3조 위반 (인간 승인 필요)**
   - 모든 거래에는 사람의 최종 승인이 필요함
   - 자동 실행 불가
