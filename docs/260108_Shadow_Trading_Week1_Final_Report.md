## 🎓 Lessons Learned

### 1. AI의 보수적 접근
**관찰**: 80% 이상 현금 보유는 과도하게 신중함
**원인 분석**:
- Hard Rules의 Agent 불일치 기준(75%)이 높아 거래 기회 제한
- Deep Reasoning Agent가 지정학적/매크로 리스크를 과대평가 가능성
- 초기 데이터 부족으로 Analyst Agent가 소극적 판단

**개선 방향**:
- Hard Rules 기준을 60%로 완화 검토
- Analyst Agent에 더 많은 데이터 소스 연결 (뉴스, 매크로 지표)

### 2. 포지션 크기 조정의 중요성
**성공 사례**: PM Agent가 Trader Agent의 제안을 REDUCE_SIZE로 조정
**효과**: 리스크 노출 감소, 청산 여유 확보
**교훈**: Risk Agent와 PM Agent의 거부권이 효과적으로 작동

### 3. Stop Loss 설정의 일관성
**관찰**: 모든 포지션에 Stop Loss 설정 완료
**평가**: ✅ 우수
**표준**: 진입가 대비 -2% ~ -5% 수준

### 4. 분할 진입 전략
**사례**: AAPL 10주 + 10주 (3분 간격)
**장점**: 평균 진입가 최적화 시도
**단점**: 거래 비용 증가 가능성 (수수료 2회)

### 5. Deep Reasoning의 영향
**Day 4-7 거래 중단 원인 추정**:
- Deep Reasoning Agent가 베네수엘라 위기, 미국 금리 등 매크로 리스크 감지
- "지루한 공방(Stagnant)" vs "새로운 위협(Escalating)" 구분 필요
- 뉴스 과잉 반응 vs 선행 지표로서의 가치 균형 필요
