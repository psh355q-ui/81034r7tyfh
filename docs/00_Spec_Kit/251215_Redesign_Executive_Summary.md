## 🏛️ 핵심 개념

### 1. 3권 분립 아키텍처

```
backend/
├── constitution/    # 입법부 (불변의 헌법)
├── intelligence/    # 사법부 (AI 토론)
└── execution/       # 행정부 (기계적 실행)
```

**원칙**:
- AI는 헌법을 읽을 수만 있고 수정 불가
- Intelligence는 제안만, Execution은 승인된 것만
- 의존성 방향: constitution ← intelligence ← execution

---

### 2. Multi-Agent Debate (War Room)

```
[Trader 🧑‍💻]: "지금 사야 합니다!"
[Risk 👮]: "안 됩니다. VIX 22, 위험합니다"
[PM 🤵]: "Risk의 의견을 채택. HOLD"
```

**핵심**:
- PM은 새로운 논리 창조 금지 (ID-Based Selector)
- 확신도의 떨림 제거, 확정된 이견만 기록
- 카카오톡 스타일 UI로 시각화

---

### 3. Shadow Trade (그림자 거래)

**기능**: AI가 거부한 제안을 가상으로 추적

**목적**: "안 샀기 때문에 손실 피했다" 증명

**리포트**:
```
🛡️ 방어 성공: 테슬라 매수 거부
→ 현재 -5.7% 하락
→ 방어한 금액: -₩57,000
```

**주의**: 수익률에 합산 금지, 방어 손실만 표시

---

### 4. Commander Mode (승인 매매)

**현재**: AI가 자동 실행  
**제안**: AI는 제안, 사용자가 승인

```
텔레그램 알림:
"엔비디아 매수 제안 상정"
[승인] [거부]
```

**메시지**:
```
"AI는 참모이고, 당신은 지휘관입니다.
최종 실행권은 오직 당신에게만 있습니다."
```

---

### 5. Trust Mileage & Circuit Breaker

**Trust Mileage**:
- AI 방어 성공 누적 → 위임 한도 확대
- 사용자 승인 필수

**Circuit Breaker**:
- VIX 급등 시 자동 발동
- Trust Mileage 즉시 회수
- 사용자 동의 불필요 (헌법적 의무)

---

### 6. KPI 재정의

**기존**:
- ❌ 총 수익률
- ❌ 수익 금액

**신규**:
- ✅ 자본 보존율: 98.5%
- ✅ 방어한 손실: -₩1,200,000
- ✅ Max Drawdown: -0.01%
- ✅ 스트레스 비교: 시장 🌊 vs 내 계좌 ⎯

**리포트**: Shield Report (방패 보고서)
