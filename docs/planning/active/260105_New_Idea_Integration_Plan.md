# 신규 아이디어 통합 계획: 이벤트 벡터 & 지정학적 리스크 점수
**날짜**: 260105
**상태**: 초안 (DRAFT)

## 1. 개요
본 계획은 `260104_chatgptideas.md`와 `260104_geminiideas.md`에서 도출된 고급 전략 아이디어를 AI 트레이딩 시스템에 통합하는 것을 목표로 합니다. 핵심은 정성적인 RSS 뉴스 데이터를 정량적인 **"이벤트 벡터(Event Vector)"**와 **"지정학적 리스크 점수(GRS)"**로 변환하여 자동화된 트레이딩 의사결정을 내리는 것입니다.

## 2. RSS 확장 (완료됨)
정보 수집 범위를 대폭 확장하여 20개 이상의 전문 피드를 추가했습니다:
- **금융 (Finance)**: FT, Nasdaq, Yahoo, Benzinga (속보성)
- **지정학 (Geopolitics)**: WTO, BBC, Defense News (맥락 파악)
- **에너지 (Energy)**: OilPrice.com, EIA, Google News Custom (심층 섹터)

## 3. 핵심 기능: 이벤트 벡터 시스템 (Event Vector System)
`DeepReasoningAgent`에 비정형 뉴스를 구조화된 벡터로 변환하는 논리 레이어를 구현합니다.

### 3.1 데이터 구조 (JSON)
```json
{
  "event_type": "군사적 긴장 고조 | 경제 제재 | 에너지 공급 충격",
  "region": "라틴 아메리카",
  "actors": ["미국", "베네수엘라"],
  "action": "해상 봉쇄 (가정)",
  "severity": 4,  // 1-5 척도
  "confidence": 0.8, // 0.0-1.0 (출처 신뢰도)
  "market_channels": ["원유", "방산", "신흥국 환율"],
  "time_horizon": "단기"
}
```

### 3.2 프롬프트 로직
- **입력**: RSS 기사 본문
- **작업**: 이벤트 유형 분류, 심각도/확신도 할당, 시장 영향 채널 식별.
- **규칙**: "단순 발언(Rhetoric)"과 "실제 행동(Action)"을 엄격히 구분 (말 < 행동).

## 4. 핵심 기능: 지정학적 리스크 점수 (GRS)
트레이딩 강도를 결정하기 위한 종합 점수 모델입니다.

**공식**:
$$ GRS = 심각도(Severity) \times 확신도(Confidence) \times 시장노출(Exposure) \times 지속성(Duration) $$

- **심각도 (1-5)**: 1=루머, 3=제재, 5=전쟁
- **확신도 (0-1)**: 0.5=단일 출처, 0.9=복수의 메이저 언론
- **시장노출 (1-3)**: 1=국지적, 3=글로벌 에너지 충격
- **지속성 (0.8-1.3)**: 0.8=단기, 1.3=장기 구조적 변화

**임계값 (Thresholds)**:
- **0-3**: 무시 (Ignore)
- **3-6**: 관찰 (Watchlist - 경계)
- **6-9**: 리스크 축소 (Hedge)
- **9+**: 리스크 오프 (청산/숏 포지션)
